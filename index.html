    Intro

    FreeReg allows you search baptism, marriage and burial records for free.

    These records have been transcribed from parish and non-conformist registers in the UK. The project is a companion project to FreeBMD (a database of the GRO birth, marriage and death indexes from 1837) and FreeCEN (a database of census information).

    We're looking for volunteers

    FreeReg is a volunteer run project. We’re always looking for new people to help transcribe data and support the running of the project.

    Make a donation

    FreeReg is exactly that, Free. As a registered charity, we don’t charge to use this site. However, FreeBMD costs money to run so please consider making a small donation.
